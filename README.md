# Bill Counter Full-Stack Project

## Overview
The **Bill Counter** is a full-stack web application designed to manage and calculate bills efficiently.  
It provides a user-friendly interface to add products, manage quantities, and automatically calculate totals.  

The project demonstrates a **React frontend** integrated with a **Spring Boot backend**, showcasing modern web development best practices.

---

## Features
- Add, edit, and delete products.
- Automatic bill calculation based on product price and quantity.
- Responsive UI built with **React** and **Vite**.
- Backend REST API built with **Spring Boot**.
- Data persistence with **H2 Database** (can be replaced with other relational databases).
- Sample data loaded on application startup.
- Clean project structure for both frontend and backend.

---

## Tech Stack
- **Frontend:** React, JavaScript, HTML5, CSS3, Vite
- **Backend:** Java, Spring Boot, Spring Data JPA
- **Database:** H2 Database (in-memory)
- **Build Tools:** Maven, npm
- **Version Control:** Git, GitHub
